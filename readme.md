# Laravel REST API

## How to install

Install dependencies via Composer

    composer install

Create and configure your *.env* file

Generate **APP_KEY**

    php artisan key:generate

Generate **JWT_SECRET**

    php artisan jwt:secret

Migrate tables

    php artisan migrate --seed


## Usage

Start local server

    php artisan serve

Enter a correct path... and use it!
